Choose data file

TestBuddy needs a file to store your user data. To choose a file simply click “choose” and create a .json file with any name, for example “test-buddy-data.json”. If you already created a file choose that file.

choose

Why

Unit data takes up far more storage space than Chrome extensions are meant to have. You can go to “About” for more information on why TestBuddy currently works the way it does.
